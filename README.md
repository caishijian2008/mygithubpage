# caishijian2008.github.io
Github Page 博客

## 安装与使用
* git clone https://github.com/caishijian2008/mygithubpage.git Hexo
* cd Hexo
* npm install
* hexo clean
* hexo generate  /  hexo g
* hexo deploy  /  hexo d
* 浏览器访问 https://caishijian2008.github.io/

## 发布文章
```
hexo new [layout] <title>
```
eg.
```
hexo new post 'My First Post'
```
